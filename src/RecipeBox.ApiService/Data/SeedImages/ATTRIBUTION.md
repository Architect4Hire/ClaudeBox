# Seed image attribution

The 31 photographs in this directory are the seed recipes' images. Every one is from
[Wikimedia Commons](https://commons.wikimedia.org/) under a licence permitting reuse, downscaled to
960px wide via Commons' own thumbnailer. They are committed rather than fetched at runtime so a fresh
`aspire run` needs nothing but the local containers — see `RecipeSeeder`, which uploads them into the
`uploads` blob container on first run.

**This file is a licence obligation, not a courtesy.** Most of these are CC BY or CC BY-SA, which
require attribution to the author. If you add, replace, or remove a seed image, update the table.

Licences present: CC BY 2.0, CC BY 4.0, CC BY-SA 2.0, CC BY-SA 3.0, CC BY-SA 4.0, CC0.
CC BY-SA images are share-alike; that obligation attaches to the image, not to this repository's code.

Each row's *Source* links to the Commons file page, which carries the authoritative licence and
author information.

## Notes

- **`apple-crumble.jpg`** is filed on Commons as "Apple Crisp" — the same dish under its American
  name. It's the weakest picture of the set (correct, but brown and flatly lit). Commons' better-lit
  alternatives were either too low-resolution for the detail hero, or were apple crumble *tart* —
  pastry-based, and so not this recipe.
- **`vegetable-lasagne.jpg`** is an aubergine lasagne, chosen because it's genuinely vegetarian;
  Commons' plain "vegetable lasagne" photos are mostly mid-assembly or unlit.
- **`chocolate-brownies.jpg`** has no author recorded in the Commons metadata. The licence is stated
  clearly on the file page; follow the link before reusing it elsewhere.
